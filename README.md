
# binks

![PyPI](https://img.shields.io/pypi/v/binks)
![APM](https://img.shields.io/apm/l/binks)

        

> A simple WSGI web server.

## Prerequisites

* Python >= 3.6
* werkzeug_raw

## Install

```sh
pip install binks
```

## Usage

```sh
binks --host 127.0.0.1 --port 8080 -w 5 --app example.wsgi:app
```
